About...

Thaumcraft 4 is a mod by Azanor for Minecraft versions 1.6.0 through 1.7.10 using the Minecraft Forge modding API.

You can download Thaumcraft 4 and other versions on CurseForge or via Azanor's official MinecraftForum thread.

Icons used for Thaumcraft's aspects were created by Lorc and Delapouite, available via Game-icons.net.

Minecraft font created by Jacob Debono.

This site's WebGL effects are assisted by TWGL.

Special thanks to webglfundamentals.org for guiding me in learning the basics of WebGL.
